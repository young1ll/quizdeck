Synology as VPC — 플랫폼 다이어그램

개인 Synology를 장기 private cloud로. quizdeck는 첫 워크로드. 학습 + 실용을 GitOps로 잇고, 플랫폼은 워크로드 위에서 진화시킨다. 텍스트 정본: platform-roadmap.md

2 · 진화 로드맵 — 워크로드 먼저, 층은 정당화하며

L1 · k3s 클라우드네이티브 substrate (Traefik 기본 번들)
기반
L2 · Cloudflare Tunnel 경계/IGW · 인바운드 0 · 홈 IP 은닉
지금
L3 · quizdeck + GitOps 첫 워크로드: Next 매니페스트 + 외부 postgres, Argo CD ← 강제 함수
지금
L4 · 관측성 Prometheus + Grafana + Loki — mesh보다 먼저
다음
L5 · service mesh Linkerd — mTLS·골든메트릭. 서비스가 늘거나 학습 모듈로
나중
🕸️
L6 · 관리 플레인 Meshery — Designs·Kanvas로 L1~L5 설계·운영·벤치마킹
나중
규율: 각 층은 독립적으로 가치 있고, 가치 < 비용 지점에서 멈춘다. Meshery는 버리는 게 아니라 맨 위 자리 — 관리할 클러스터가 생겼을 때 빛난다.
